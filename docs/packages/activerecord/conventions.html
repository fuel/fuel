ActiveRecord

ActiveRecord is a package that allows you to use the Active Record Pattern in your application Models.

Conventions

Unlike most Fuel classes, ActiveRecord has some conventions. This is to ease its use.

Entends

Your models must extend the ActiveRecord\Model class.

class Model_Entry extends ActiveRecord\Model { }

File Locations

The Model class files can be located either in classes/model/ or simply in classes/. The placement of the file determines the name of the class:

// File in classes/model/entry.php
class Model_Entry extends ActiveRecord\Model { }

// File in classes/entry.php
class Entry extends ActiveRecord\Model { }

It is up to you what to use, but we recommend using the model directory as this keeps all your models in one place.

Please Note: These are the only 2 paths that models can be located.
