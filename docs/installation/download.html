Contribute to the core

FuelPHP is developed using a community approach and we encourage participation. If you are interested in contributing please take a second to read our Coding Standards. Any contributions that do not meet these guidelines will not be accepted. Otherwise: fork, commit & make pull-request. If you don't know how: http://help.github.com/forking/
