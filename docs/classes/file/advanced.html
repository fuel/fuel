File Class

The file class offers a set of methods to working with files & directories. This can be done through some helper methods or through a more advanced object oriented method where all files and directories are wrapped in objects.

Advanced usage

Aside from the basic static usage of the File class there are also more complex options available. The first are Area's which are basicly a basepath it cannot act outside of and some configuration options. And when using the File class's methods on an area they will return objects wrapping directories & files.
