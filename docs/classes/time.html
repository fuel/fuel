Time Class

The Time class is a set of helper functions for generating and managing time related operations..

offset($remote, $local = NULL, $now = NULL)

The offset method returns the offset (in seconds) between two time zones. Use this to display dates to users in different time zones.

Static Yes
Parameters
Param Default Description
$remote
Timezone that to find the offset of
$local 
null
 Timezone used as the baseline
$now 
null
 UNIX timestamp or date string
Returns integer
Example 
$seconds = Time::offset('America/Chicago', 'GMT');

 End value
Returns array

months($format = NULL)

The months method returns the Number of months in a year. Typically used as a shortcut for generating * a list that can be used in a form.

Static Yes
Parameters
Param Default Description
$format 
null
 The format to use for months
Returns array
Example 
By default a mirrored array of $month_number => $month_number is returned

	$months = Time::months();
	// aray(1 => 1, 2 => 2, 3 => 3, ..., 12 => 12)

But you can customise this by passing in either Time::MONTHS_LONG

	$months = Time::months(Time::MONTHS_LONG);
	// array(1 => 'January', 2 => 'February', ..., 12 => 'December')

Or Time::MONTHS_SHORT

	$months = Time::months(Time::MONTHS_SHORT);
	// array(1 => 'Jan', 2 => 'Feb', ..., 12 => 'Dec')

years($start = NULL, $end = NULL)

The years method returns an array of years between a starting and ending year. By default, the the current year - 5 and current year + 5 will be used. Typically used as a shortcut for generating a list that can be used in a form.

Static Yes
Parameters
Param Default Description
$start 
null
 starting year (default is current year - 5)
$end 
null
 ending year (default is current year + 5)
Returns array
Example 
$years = Time::years(2000, 2010);

span($remote, $local = NULL, $output = NULL)

The span method returns a new date object.

Static Yes
Parameters
Param Default Description
$remote
timestamp to find the span of
$local 
null
 timestamp to use as the baseline
$output 
null
 formatting string
Returns string when only a single output is requested
array associative list of all outputs requested
Example 
$span = Time::span(60, 182, 'minutes'); // 2
$span = Time::span(60, 182, 'minutes,seconds'); // array('minutes' => 2, 'seconds' => 2)

fuzzy_span($timestamp, $local_timestamp = NULL)

The fuzzy_span method returns the difference between a time and now in a "fuzzy" way. Displaying a fuzzy time instead of a date is usually faster to read and understand.

Static Yes
Parameters
Param Default Description
$timestamp
"remote" timestamp
$local_timestamp 
null
 "local" timestamp, defaults to time()
Returns string
Example 
$span = Time::fuzzy_span(time() - 10); // "moments ago"
$span = Time::fuzzy_span(time() + 20); // "in moments"
