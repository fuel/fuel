Agent Class

The agent class is a set of methods to help with easy access to user agent information.

string()

The string method returns the user agent string.

Static Yes
Returns string
Example 
if ($agent_string = \Agent::string())
{
	// do something with $agent_string
}

Returns string

is_browser($browser = null)

The is_browser method checks if the user agent is a browser. If the browser param is given, that given browsername will be checkt against the user agent.

Static Yes
Parameters
Param Default Description
$browser null A string with the browsername (or array with browsernames) will be checkt against the user agent.
Returns bool
Example 
if (\Agent::is_browser())
{
	// do something special
}

if (\Agent::is_browser('MSIE')) // case sensitive
{
	// it's Internet Explorer, check version and if not 9 apply html5shiv
}

if (\Agent::is_browser(array('Firefox', 'Chrome', 'Opera'))) // case sensitive
{
	// it's all fine, need to do nothing
}

is_robot($robot = null)

The is_robot method checks if the user agent is a robot. If the robot param is given, that given robotname will be checkt against the user agent.

Static Yes
Parameters
Param Default Description
$robot null A string with the robotname (or array with robotnames) will be checkt against the user agent.
Returns bool
Example 
if (\Agent::is_robot())
{
	// load robot view
}
else
{
	// load normal view
}

if (\Agent::is_robot('googlebot'))
{
	// it's Google Bot
}

if (\Agent::is_robot(array('slurp', 'yahoo')))
{
	// it's Yahoo
}
