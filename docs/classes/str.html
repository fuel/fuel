Str Class

The str class is a set of methods to help with the manipulation of strings.

alternator(*args)

The alternator method allows two or more items to be alternated between, when cycling through a loop.

Static Yes
Parameters Unlimited string arguments
Returns void
Example 
for ($i = 0; $i < 10; $i++)
{
    echo Str::alternator('one', 'two', 'three', 'four', 'five');
    echo "\n";
}
/* Outputs
one
two
three
four
five
one
two
three
four
five
*/

quotes_to_entities($str)

The quotes_to_entities converts single and double quotes to html entities.

Static Yes
Parameters
Param Default Description
$str required The string containing the quotes to convert.
Returns string
Example 
$str = "Bob's mother said, \"I like pie\"."
$output = Str::quotes_to_entities($str);
// Returns Bob&#39;s mother said, &quot;I like pie&quot;.

random($type = 'alnum', $length = 8)

The random method generates a random string based on the type given.

Static Yes
Parameters
Param Default Description
$type alnum The type of string to generate. Your choices are alnum, numeric, nozero, alpha, distinct, hexdec, unique, and sha1.
$length 
16
 The number of characters you would like the final string to be (unique and sha1 ignore this parameter).
Returns string
Example 
// alnum (uppercase and lowercase letters mixed with numbers)
Str::random('alnum', 16);
// Returns: SvZi9Dh3lq7zQYim
 
// numeric (just numbers)
Str::random('numeric', 16);
// Returns: 1045343964672481
 
// nozero (just numbers excluding zero)
Str::random('nozero', 16);
// Returns: 3244623373994515
 
// alpha (just uppercase and lowercase letters)
Str::random('alpha', 16);
// Returns: LuVAXbmxQbbWoYqz
 
// distinct (uppercase letters and numbers that cannot be confused)
Str::random('distinct', 16);
// Returns: R79MPKMH4KTRN35J
 
// hexdec (hexadecimal characters a-f, 0-9)
Str::random('hexdec', 16);
// Returns: 09c34e42f36547f8
 
// unique (32 character string based on md5)
Str::random('unique');
// Returns: ed4bb844a35b7a4edb7eed0d3795d328
 
// sha1 (40 character string based on sha1)
Str::random('sha1');
// Returns: af5c5a8cc3be9a3180205c1ed2975015cd6cf1e7

reduce_double_slashes($str)

The reduce_double_slashes method converts double slashes in a string to a single slash, except those found in http://.

Static Yes
Parameters
Param Default Description
$str required The string to reduce double slashes.
Returns string
Example 
$string = "http://example.com//index.php";
echo Str::reduce_double_slashes($string); 
// Result "http://example.com/index.php"

reduce_multiples($str, $character = ',', $trim = false)

The reduce_multiples method reduces multiple instances of a particular character.

Static Yes
Parameters
Param Default Description
$str required The string to search
$character 
,
 The character to search for multiples of
$trim 
false
 If true the character will be trimmed off of the ends of the string
Returns string
Example 
$str = 'Bob, Sally, Tim,, Jake,';
echo Str::reduce_multiples($str, ',', true);
// Result "Bob, Sally, Tim, Jake"

repeater($data, $num = 1)

The repeater method repeats a character or string n times.

Static Yes
Parameters
Param Default Description
$str required The string or character to repeat
$num 
1
 The number of times to repeat the string
Returns string
Example 
echo Str::repeater('\n', 5);
// Result \n\n\n\n\n

strip_quotes($str)

The strip_quotes method removes single and double quotes from a string.

Static Yes
Parameters
Param Default Description
$str required The string to manipulate
Returns string
Example 
$str = "Bob's mother said, \"I like pie\".";
echo Str::strip_quotes($str);
// Result: Bobs mother said, I like pie.

strip_slashes($str)

The strip_slashes method removes slashes contained in a string or in an array.

Static Yes
Parameters
Param Default Description
$str required The string or array to manipulate
Returns string
Example 
$str = 'O\'Reilly';
echo Str::strip_slashes($str);
// Result: O'Reilly

$arr = array('O\'Reilly', 'O\'Brien');
$arr = Str::strip_slashes($arr);
// Result: array(O'Reilly, O'Brien)

trim_slashes($str)

The trim_slashes method removes any leading/trailing slashes from a string.

Static Yes
Parameters
Param Default Description
$str required The string to be manipulated
Returns string
Example 
echo Str::trim_slashes("/url/to/your/post/");
// Result: url/to/your/post
